## Introduction

In Episode 54 of the SAP on Azure Video Podcast we talk about Application Security Groups, ASUG & DSAG Study, Fusion Detection for Ransomware, SAP Event Mesh & Azure Event Grid integration, SAP Gateway and Azure API Management and OData Filter and Selects with Synapse Pipelines. Then we have Chetan Desai talking about the new SAP Implementation Design Principles for SAP SuccessFactors and Azure Active Directory.
