## Introduction

In Episode 60 of the SAP on Azure Video Podcast we talk about remote audits of Azure Datacenters using HoloLens 2, General availability of Oracle consistent snapshots using Azure VM Backup, Creating Power Automate flows with Visio, Integrating Teams with SAP Digital Manufacturing Cloud, Private preview for Configuration & SAP Installation for the SAP Deployment Auto and scaling your SAP CPI instance with Azure. Then Bernhard Friess joins us to talk about the SAP on Azure German v-Team.
