## Introduction

In Episode 61 of the SAP on Azure Video Podcast we talk about the upcoming openSAP course for SAP and Microsoft integration, Azure credits for open source projects, the European Eclipse Data connector, Trusted cloud principles, Azure Purview going GA, Azure virtual Datacenter tour with SAP, Microsoft Research Summit, Microsoft ignite, abapConf 2021 and Steampunk. Then Holger Bruchelt talks about some preview that are currently available for our customers and partners in the context of SAP and Microsoft.
